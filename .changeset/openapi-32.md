Add OpenAPI 3.2 support across version detection, reference resolution, linting, bundling, and language-server completion/symbol features. This includes `$self`, `query` and `additionalOperations`, reusable media types and sequential encoding, expanded examples and security schemes, Security Requirement URI references, and the new 3.2 metadata fields.
